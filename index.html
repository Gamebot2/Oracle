Answer and Context

Oracle allows you to submit "questions to documents". You can either get the direct answer to your question, or you can get some context lines to find the answer yourself. E.g, you might ask a contract "How many preferred shares are listed?".

BERT and BM25

In the backend, Oracle uses the Bert model for answer queries and the BM25 model for context queries. You can find more information about the bert model here.

Clean UI

My main contribution was the UI, built with ReactJS on the Instabase stack. You can ask questions and get contexts asynchronously, and export to CSV or JSON when you're done. It also displays the document for your convenience.
